 
\CWHeader{Лабораторная работа \textnumero 9}

\CWProblem{
	Поиск компонент связности
	
	Задан неориентированный граф, состоящий из $n$ вершин и $m$ ребер. Вершины пронумерованы целыми числами от $1$ до $n$. Необходимо вывести все компоненты связности данного графа.
	
	{\bfseries Формат ввода:} 
	В первой строке заданы $1 \leq n \leq 10^5$ и $1 \leq m \leq 10^5$. В следующих m строках записаны ребра. Каждая строка содержит пару чисел – номера вершин, соединенных ребром. 
	
	{\bfseries Формат вывода:} Каждую компоненту связности нужно выводить в отдельной строке, в виде списка номеров вершин через пробел. Строки при выводе должны быть отсортированы по минимальному номеру вершины в компоненте, числа в одной строке также должны быть отсортированы.  
}
